# odin-recipes

This is the first project assigned by TheOdinProject. It's a project built entirely with HTML, and it's basic tags.

This website allows the user to view the recipes for three foods that I've chosen. The website includes three webpages - corresponding to three recipes. The webpages are all stored in a directory named 'recipes'.

The links for the webpages in the landing page is arranged as an unordered list to make sure that they aren't arranged in a straight line. Clicking on each link will take you to the corresponding webpage for the recipe.

The webpage for the recipe contains the recipe title, description, image, ingredients required, and steps required. All these segments are separated by varying heading sizes. The images are chosen from a directory named 'images'.

The important segments of the recipes are highlighted accordingly.
